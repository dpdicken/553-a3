Introduction:
This is a tutorial on how to use the profiling tool kcachegrind. This tool was developed as an add on to valgrind. It uses valgrind's cachegrind utility, which profiles cache lines running in valgrinds shadow architecture.
Within this tutorial, you will set up kcachegrind and run it on the lulesh application.
